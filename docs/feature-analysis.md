# Plot3D 功能分析与优化建议

> 从材料和统计科学家角度分析科研绘图软件的功能完善性

---

## 📊 现有功能总结

### 图表类型（10种）
| 类别 | 图表类型 |
|------|----------|
| 2D | 线图(line)、散点图(scatter)、柱状图(bar)、面积图(area)、饼图(pie)、极坐标图(polar) |
| 3D | 曲面图(surface3d)、散点图(scatter3d)、等高线图(contour3d)、柱状图(bar3d) |

### 曲线拟合功能
- **线性拟合**：斜率、截距、R²、置信区间
- **多项式拟合**：2-6阶，含完整统计信息
- **非线性拟合**：指数、对数、幂函数、高斯、逻辑斯蒂
- **统计输出**：R²、调整R²、RMSE、MAE、残差标准误差、参数标准误差、95%置信区间
- **残差分析**：残差图（vs拟合值/vs X值）

### 数据变换功能
| 类别 | 操作 |
|------|------|
| 数学变换 | ln、lg、exp、√、平方、倒数、绝对值 |
| 三角变换 | sin、cos、tan、角度→弧度 |
| 列运算 | 加、减、乘、除（X与Y列） |
| 数据操作 | 升序/降序排序、归一化、添加列/行 |

### 数据导入导出
- **导入**：CSV、Excel (.xlsx/.xls)
- **导出**：PNG、SVG、PDF、TIFF、CSV
- **项目文件**：.plot3d格式（保存完整项目状态）

### 其他功能
- **注释**：文本、箭头、矩形、LaTeX公式
- **图层管理**：多数据集叠加、样式配置、误差棒
- **颜色映射**：jet、viridis、hot、coolwarm、parula、plasma、cividis、inferno、magma、turbo、batlow
- **数据生成**：正弦函数、sinc曲面、球体、柱状图示例数据

## 🔬 缺失的关键功能

### 1. 统计分析模块 ⭐⭐⭐（高优先级）

科研绘图软件缺少基础统计分析功能：

#### 描述性统计
- 均值 (Mean)
- 标准差 (Standard Deviation)
- 方差 (Variance)
- 中位数 (Median)
- 四分位数 (Quartiles, Q1/Q2/Q3)
- 偏度 (Skewness)
- 峰度 (Kurtosis)
- 范围 (Range, Min/Max)

#### 分布检验
- Shapiro-Wilk 正态性检验
- Kolmogorov-Smirnov 检验
- Q-Q图绘制

#### 假设检验
- t检验：单样本、双样本、配对样本
- ANOVA：单因素、多因素
- 非参数检验：Mann-Whitney U、Wilcoxon signed-rank、Kruskal-Wallis

#### 相关性分析
- Pearson 相关系数
- Spearman 秩相关
- Kendall 相关
- p值计算
- 相关性矩阵可视化

#### 回归诊断
- Durbin-Watson 检验（自相关）
- Breusch-Pagan 检验（异方差）
- VIF 计算（多重共线性）
- 残差正态性检验

---

### 2. 误差棒增强 ⭐⭐⭐

当前支持误差棒，但功能不完整：

#### 缺失功能
- **置信区间误差棒**：95% CI 自动计算
- **标准误差棒**：SE vs SD 区分显示
- **不对称误差棒**：可视化优化
- **误差棒统计计算**：从原始数据自动计算 SE/SD/CI
- **误差棒样式**：caps样式、宽度配置

#### 建议实现
```typescript
interface ErrorBarConfig {
  type: 'sd' | 'se' | 'ci95' | 'custom';
  capWidth: number;
  capStyle: 'line' | 'bracket';
  showCap: boolean;
  asymmetric: boolean;
}
```

### 3. 多峰拟合 ⭐⭐⭐

材料科学核心需求（XRD谱图、光谱分析）：

#### 功能需求
- **多峰Gaussian拟合**：自动识别多个峰
- **峰识别与定位**：自动检测峰位置
- **峰面积积分**：计算各峰面积
- **峰型拟合**：Voigt、Lorentzian、Pseudo-Voigt
- **背景扣除**：线性/多项式背景拟合
- **峰参数导出**：位置、高度、宽度、面积

#### 应用场景
- XRD物相分析
- Raman/IR光谱分析
- XPS谱图拟合
- PL光谱分析

## 🎯 总结

Plot3D 作为科研绘图软件，已具备基础的绘图和拟合功能，但在统计分析、数据处理、专业图表类型方面仍有较大提升空间。建议优先实现：

1. **统计分析模块** —— 科研数据分析的基础
2. **多峰拟合** —— 材料科学核心需求
3. **箱线图/热力图** —— 常用缺失图表
4. **拟合结果导出** —— 提升工作效率

这些功能将显著提升软件在科研领域的实用性和竞争力。
